# Motion Detector

Motion detector with PIR sensor.

# Parts

- PIR motion sensor
- LED
- Resistors(100 ohm)
- Breadboard
- Jumper wires

# Schematic

![Schematic](schematic/schematic.png)

# Instructions

To start this app:

- Installing Packages

```
pio pkg install
```

- Build Projects

```
pio run
```

- Upload firmware to device:

```
pio run --target upload
```
