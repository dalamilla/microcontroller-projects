# Dice Roller

Dice Roller with leds.

# Parts

- LED x 7
- Resistors(100 ohm) x 7
- Breadboard
- Jumper wires

# Schematic

![Schematic](schematic/schematic.png)

# Instructions

To start this app:

- Installing Packages

```
pio pkg install
```

- Build Projects

```
pio run
```

- Upload firmware to device:

```
pio run --target upload
```
