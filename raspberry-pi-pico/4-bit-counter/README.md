# 4-Bit Counter

4-Bit counter  with leds.

# Parts

- LED x 4
- Resistors(100 ohm) x 4
- Breadboard
- Jumper wires

# Schematic

![Schematic](schematic/schematic.png)

# Instructions

To start this app:

- Installing Packages

```
pio pkg install
```

- Build Projects

```
pio run
```

- Upload firmware to device:

```
pio run --target upload
```
