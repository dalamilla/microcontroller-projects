# Holiday Lights

Holiday lights patterns.

# Parts

- LED x 6
- Resistors(100 ohm) x 6
- Push Button
- Breadboard
- Jumper wires

# Schematic

![Schematic](schematic/schematic.png)

# Instructions

To start this app:

- Installing Packages

```
pio pkg install
```

- Build Projects

```
pio run
```

- Upload firmware to device:

```
pio run --target upload
```
